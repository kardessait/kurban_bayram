# 🌙 Kurban Bayramı Kutlama Sitesi — v1.0.0

Kurban Bayramı'nı kutlamak için tasarlanmış, tamamen saf HTML/CSS/JS ile yazılmış animasyonlu bir web sayfası.

## ✨ Özellikler

- 🐑 Canvas üzerinde yürüyen sevimli animasyonlu koyunlar (göz kırpma dahil)
- 🌙 Hilal ve yıldız efekti
- ✨ Neon ışıklı "Kurban Bayramınız Kutlu Olsun" yazısı — yavaş fade-in animasyonuyla
- 🏮 Sallanan renkli fenerler (kırmızı, altın, yeşil)
- ⭐ Titreşen yıldızlı gece gökyüzü
- 🌿 Animasyonlu yeşil zemin

## 🛠️ Kullanılan Teknolojiler

- Vanilla HTML5 / CSS3 / JavaScript
- HTML5 Canvas API (koyun animasyonları)
- CSS Keyframe Animations
- Sıfır bağımlılık — framework yok, kütüphane yok

## 🚀 Kullanım

Hiçbir kurulum gerektirmez. `kurban_bayrami.html` dosyasını indirip tarayıcıda açmanız yeterli.

```bash
git clone https://github.com/kullanici-adi/kurban-bayrami
cd kurban-bayrami
# kurban_bayrami.html dosyasını tarayıcıda aç
```

## 📁 Dosya Yapısı

```
kurban-bayrami/
└── kurban_bayrami.html   # Tek dosya, her şey içinde
```
